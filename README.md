   ## Installation
   
   
 * Run on the terminal:<br/>
    ```
    $ mysql -u root -p
    ```

  * Run the following to create the Database ecommerce_db:<br/>
    ```
    $ source db/schema.sql
    ```

  * Run the following to create and seeded the tables: <br/>
    ```
    $ npm run seed
    ```
    
  * Start the server with: <br/>
    ```
    $ npm start
    ```
