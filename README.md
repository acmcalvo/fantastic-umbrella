  ## Technologies Used
  
  * npm 
  * npm seed
  * npm bcrypt
  * npm dotenv to store environmental variables.
  * npm  mysql2
  * heroku to run jawsDBMySQL

   ## Installation
   
   * run
    ```
  $ mysql -u root -p
    ```

  * Run the following to create the Database ecommerce_db:
    ```
    $ source db/schema.sql
  ```

  * Run the following to create and seeded the tables:
    ```
 $ npm run seed
   ```
